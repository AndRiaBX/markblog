# Hello, MarkBlog!

Welcome to **MarkBlog** — a lightweight blog engine that renders Markdown files.

## Features

- Write posts in **plain Markdown**
- **YAML frontmatter** for metadata
- **Tag-based** navigation
- **RSS feed** auto-generated
